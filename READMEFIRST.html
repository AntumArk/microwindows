Basic Instructions for Building, Installing, and Executing Microwindows under X11

------------------------------------------------------------------------------------------------------------

After downloading the Microwindows package, unpack it into a suitable directory. This is usually a new one under
your personal or home directory on your Linux system.


For a Quickbuild, you can follow these instructions, for further details and the Latest FAQ, please read
the /doc/faq2.

Open up a Terminal or Console and go into the "microwindows/src" directory.

There is no need to run ./configure with Microwindows, it's makefile is smart enough and builds everything locally.

Enter: "make".

Make will now build the Microwindows libraries and the demo applications for you, and in the "src/lib" directory
you will now find the newly generated libraries and in the "src/bin" directory the newly compiled demo programs.

Then enter "make install".

This installs the newly created files to your systems standard locations.
The Makefile does assume that you are using a Linux system with X11 installed.
For testing purposes, Microwindows should run on your X11 version and will generate shared not static libraries.

------------------------------------------------------------------------------------------------------------

To execute the demo programs run the script "rundemos.sh" in the "src" directory. This will list the
available demos and allow you to select a program to run. Before selecting a new demo program, close all
windows opened by the previous demo. There is a "demos.txt" which contains short descriptions of these demos.

These demos are based on the Nano-X and MWin apis. Demos for the X11 compatibility layer NX11
are located in the "nxlib/test" folder. You can run them in there e.g. with "./textbox". The scripts provided
in the "nxlib/test" folder can be used if Nano-X is used as a static library.

The Nano-X library and the NXlib compatibility layer library are also available linked together
to the PX11 library. The demo programs linked with that are available in the "nxlib/ptest" folder.

------------------------------------------------------------------------------------------------------------

Documentation is available in the "doc" folder. The latest information is in the "doc/faq2" file and in the
"NX11 programming manual.html", which is the X11 compatibility library for Nano-X.
Further documentation is currently dated. There is a tutorial for Nano-X and documentation regarding Nano-X in the
nano-X folder.

A reference for the Nano-X API is in the "doc" folder. The Microsoft Windows compatible MWin API is described in
the "doc/mwinapi.html" file. The MWin API implements a subset of the Microsoft Windows API.
